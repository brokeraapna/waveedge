// OUR METHODOLOGY
Gaurav's core Elliott Wave methodology uses the MACD signal line — not the MACD line — as the primary wave trigger. Bullish divergence on the signal line marks Wave 1. Zero-line crossovers mark subsequent waves. Bearish divergence at Wave 5 confirms cycle exhaustion. This 9-state framework (Idle → W1 → W2 → W3 → W4 → W5 → WA → WB → WC) underpins all platform tools, Pine Script indicators, and educational content.
